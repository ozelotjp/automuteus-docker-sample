This is the sample code to refer to when running [automuteus](https://github.com/denverquane/automuteus) on a self-hosted system.

## Requirements

- Docker
- Docker-Compose

## Steps

1. Createy `.env`
   1. Copy `.env.example`
   2. Edit `AUTOMUTEUS_TAG`, `GALACTUS_TAG`, and `DISCORD_BOT_TOKEN`
2. Exec `./start.sh`
